Bat Species Identification on Edge Device

Domain: Deep Learning, Machine learning, IoT

Identifying Bat signals in the UAE based on Audio samples by building appropriate Convolutional Neural Network (CNN) - optimized for performance and minimal power and memory consumption. Project is done in collaboration with Emirate Nature - World Wildlife Federation (WWF).

The CNN model will be deployed in an edge device, located in the mountaineous region in the UAE where bat species are typically located. The project aims to enhance wildlife monitoring and increase knowledge about bat species habitat and migration routes.

Technologies Used: Keras, Tensorflow, Pytorch

Car Pooling Website for University Students

Domain: Software Engineering

This project involved building a car pooling website where University students can register and eithe request or apply for rides. Verification features were added such as ride times. SQL database was used for data persistance. User account security features involving hashing of passwords for safe storage was implemented.

The design of the website implemented aesthethic colors and responsive elements to allow user to access from mobile devices. Forms implemented error handling for a more user-friendly experience.

Technologies Used: Java, JSP, Javascript, SQL, Bootstrap, HTML, CSS

HyperJump (Physical Game)

Domain: Software Development, Embedded Systems

This project involved the design and development of an Arduino-based sensor physical game, with Java to manage the game logistics. A Java Software was written to handle the starting and ending of the game and also manage scoreboard.

The game rules are simple, a spin-off version of the classic arcade game Jumping Jackpot. Just jump when the light passes the end of the loop. A sensor band on the player's leg detects the jump. The opponent controls how fast the light in the loop moves, and also the direction!

Technologies Used: Java, Arduino, XBee
